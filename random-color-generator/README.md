# Random Color Generator

A clean, modern web app that generates random colors and gradients.  
Built with vanilla HTML, CSS, and JavaScript – perfect for beginners and a quick reference.

![screenshot](https://via.placeholder.com/800x400?text=Random+Color+Generator)  
*(Add your own screenshot later)*

## Features

- **Random Solid Color** – click a button, change the whole page background instantly.
- **Random Gradient** – generate beautiful linear gradients with a random angle.
- **Copy to Clipboard** – copy HEX code or full CSS gradient with one click.
- **Recent Colors** – automatically saves the last 10 colors you generated.
- **Dark / Light Mode** – toggle between themes, preference is saved.
- **Responsive Design** – looks great on desktop and mobile.
- **Smooth Animations** – background and UI transitions are buttery smooth.

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Flexbox, Transitions)
- Vanilla JavaScript (ES6+)
- LocalStorage for data persistence
